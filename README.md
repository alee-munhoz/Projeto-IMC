# Relatório – Aplicativo Calculadora de IMC

## Objetivo
O objetivo deste projeto foi desenvolver um aplicativo mobile capaz de calcular o IMC (Índice de Massa Corporal) a partir das informações fornecidas pelo usuário.

## Funcionamento
O aplicativo possui duas telas. Na primeira tela, o usuário informa nome, peso e altura e pressiona o botão Calcular. Em seguida, o aplicativo realiza o cálculo automaticamente.

Na segunda tela, o sistema apresenta o valor do IMC e uma mensagem com a classificação, como abaixo do peso, peso normal, sobrepeso ou obesidade. Também existe um botão Voltar para retornar à tela inicial.

## Tecnologias Utilizadas
O projeto foi desenvolvido utilizando Java no Android Studio, com a interface criada em XML.

## Conclusão
Este projeto permitiu praticar conceitos de desenvolvimento mobile, como criação de telas, captura de dados do usuário, realização de cálculos e navegação entre atividades dentro de um aplicativo Android.
